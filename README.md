# WeatherApp

**API를 이용한 날씨앱⛅️**



✅ **주요기능**

> -Interface Builder, AutoLayout을 활용해서 UI를 구성하였습니다.
>
> -**테이를뷰**를 활용해서 날씨 데이터를 표시했습니다.
>
> -사용자에게 **위치 정보**를 요청하고 GPS에서 좌표를 얻었습니다.
>
> -날씨 **API 분석**부터 **JSON 파싱**까지 API 연동에 필요한 모든 과정을 경험했습니다.
